About YACertManager

YACertManager

Version {{ version }}
Purpose

YACertManager is a modern, web-based Certificate Authority (CA) management system designed to simplify the creation and management of internal Public Key Infrastructures (PKI). It provides a streamlined interface for managing Root CAs, Intermediate CAs, and issuing server certificates.

Key Features
  • Root & Intermediate CA Management
  • Server Certificate Issuance with SANs
  • Certificate Signing Request (CSR) Support
  • Certificate Import & Export
  • Visual Certificate Inspection
Technology Stack

Built with modern technologies for performance and reliability:

Python 3.10+ FastAPI Bootstrap 5 OpenSSL Docker
Security Notice: This application is designed for development, testing, and internal infrastructure environments. For high-security production PKI requirements, please consult security best practices and enterprise-grade solutions.
