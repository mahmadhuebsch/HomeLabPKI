Getting Started

Welcome to YACertManager!

A web-based Certificate Authority management system for creating and managing Root CAs, Intermediate CAs, and server certificates.

Quick Start Guide:

  1. Create a Root CA - The trust anchor for your PKI
  2. Create Intermediate CA - For day-to-day signing (recommended)
  3. Issue Certificates - Create server/client certificates

PKI Hierarchy:

Intermediate CA
Certificates

Best Practices:

  • Keep Root CA private key secure
  • Use Intermediate CAs for signing
  • Monitor expiring certificates
  • Use strong key algorithms
