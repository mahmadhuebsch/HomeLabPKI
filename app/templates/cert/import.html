About Certificate Import

What can I import?

Import server/client certificates (TLS, code signing, etc.) for tracking purposes.

Prerequisites

The issuing CA must already exist in the system. Import CAs first:

  1. Root CA (via Import Root CA)
  2. Intermediate CA (via Import Intermediate CA)
  3. Then import certificates here

File Format

Upload a single PEM-encoded certificate file (.crt, .pem, .cer).

Note: CA certificates cannot be imported here. Use the CA Import pages instead.

What gets stored?

  • The certificate file
  • Certificate metadata

What does NOT get stored?

  • Private key (managed externally)
Imported certificates are marked as External to indicate the private key is not managed by this system.
